# Epitech-stats-notes

```sh
 $> ./stats-notes -u prenom.nom@epitech.eu [ModuleCodes...]
```

## Options
* mode verbose ```-v boolean```
* target user  ```-u login```
* output ```-o str```

## Configuration

```sh
 $> ./stats-notes -a auth-xxxxxxxxxxxxxxx -u prenom.nom@epitech.eu [ModuleCodes...]
```
